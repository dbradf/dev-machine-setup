# Dev Machine Setup

Configuration to setup a development machine.

Currently only supporting Ubuntu.

## Usage

### Setup Steps

- [ ] Generate SSH keys
- [ ] Add public key to github
- [ ] Install `python3-pip`, `python3-apt`, `python3-venv`, and `git`
- [ ] `pip3 install wheel`
- [ ] Clone repo

### Running

Run:
```bash
$ ./bootstrap.sh
```
